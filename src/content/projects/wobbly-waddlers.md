A 24-hour hackathon game that mashes Snake, Sokoban, and a balancing arena into one. Push crates, collect eggs, and make sure you don't fall off the edge — all at the same time.

[Play it on itch.io →](https://bolin8621.itch.io/wobbly-waddlers)

---

# Behind the build

Built in 24 hours at NUS Hack&Roll 2024. The most interesting part of this game was coming up with the idea. We were inspired by many simple and different game mechanics, and thought it would be cool to see how merging them together would fair. Turns out, pretty fun!
